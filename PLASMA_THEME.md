# 🎨 AstraGPT Plasma Theme Showcase

## Visual Effects Overview

### 🌈 Color Palette
- **Primary Accent**: Hot Pink (#ff006e) - Main action buttons & highlights
- **Secondary Accent**: Cyan (#00f5ff) - Complementary elements
- **Tertiary Accent**: Purple (#b537f2) - Gradient transitions
- **Background**: Deep Space (#0a0014) - Dark base

### ✨ Plasma Effects

#### 1. Animated Background
- Multiple radial gradients creating plasma effect
- 20-second rotation and movement animation
- 60px blur for smooth color bleeding
- Colors: Pink, Cyan, Purple

#### 2. Glassmorphism
- Backdrop blur on all panels (10-20px)
- Semi-transparent backgrounds (rgba with alpha 0.4-0.8)
- Frosted glass effect throughout UI

#### 3. Glow Effects
- Logo icon: Pulsing glow (2s animation)
- Buttons: Multi-layered shadow glows
- Avatars: 15-20px colored glows
- Input focus: 30px glow spread
- Send button: Pink/purple combined glow

#### 4. Gradient Animations

**Buttons:**
- New Chat: Pink → Purple gradient
- Upgrade: Cyan → Purple gradient
- Send: Pink → Purple gradient
- Quick Actions: Custom color gradients per action

**Shine Effect:**
- Sliding white gradient overlay on hover
- 0.5s transition animation

#### 5. Transform Animations

**Hover States:**
- translateY(-2px) lift effect
- scale(1.05) slight growth
- Combined with glow increase

**Float Animation:**
- Welcome icon floats up/down (3s cycle)
- Floating particles (8-10s cycles)
- Smooth ease-in-out timing

#### 6. Particle System
- Two floating gradient orbs in main content
- 300px & 200px diameter
- Pink and cyan variants
- Infinite floating animation

#### 7. Additional Effects
- Scanline overlay (retro CRT feel)
- Border highlights on hover
- Inset glows on active states
- Smooth color transitions (0.3s)
- Pulsing typing indicator with gradients

### 🎯 Interactive Elements

#### Sidebar
- Glassmorphism background
- Purple shadow glow
- Smooth slide-in on mobile

#### Chat Header
- Blurred backdrop
- Elevated with drop shadow
- Model selector with hover glow

#### Messages
- Avatar glows (pink for user, cyan for AI)
- Fade-in animation
- Gradient background on avatars

#### Input Area
- Focus glow (pink)
- Auto-grow textarea
- Gradient send button
- Voice button gradient reveal

#### Quick Actions
- Color-coded by category
- Individual glow colors
- Shine effect on hover
- Translated lift animation

### 📱 Responsive Behavior
- Plasma effects scale appropriately
- Mobile overlay with blur
- Sidebar slide with smooth transition
- Animations optimized for performance

### 🎭 Theme Consistency
- All interactive elements glow
- Consistent color language
- Unified animation timing (0.3s standard)
- Gradient direction consistency (135deg)

## Performance Considerations
- CSS-only animations (GPU accelerated)
- Backdrop-filter with moderate blur values
- Transform animations (efficient)
- Will-change where needed
- Optimized gradient calculations

## Browser Support
Requires modern browsers with support for:
- backdrop-filter
- CSS gradients (linear & radial)
- CSS animations
- CSS transforms
- rgba colors with alpha
